![정적 사이트와 Hugo 로고](/images/image-01.jpg)
정적 사이트는 빠르고, 안전하며, 호스팅하기 쉽습니다. Hugo는 그중에서도 손꼽히는 정적 사이트 생성기입니다.

## Hugo를 고르는 이유

Hugo는 속도와 단순함을 제공합니다. 마크다운 파일에 콘텐츠를 쓰면, Hugo가 몇 초 만에 완성된 웹사이트로 만들어 줍니다.

## 시작하기

Hugo를 설치하고 새 사이트를 만든 뒤, 테마를 고르면 글을 쓰기만 하면 됩니다. GitHub Actions로 GitHub Pages에 배포하는 것도 어렵지 않습니다.

## 마치며

Hugo와 GitHub Pages로 만든 정적 블로그는 개인 블로그나 프로젝트 블로그에 잘 맞습니다. 작게 시작해서 필요할 때마다 키워 나가면 됩니다.
